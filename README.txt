This plugin adds a custom widget that displays a slideshow of Cratejoy subscription boxes. This plugin relies on https://www.cratejoy.com/ as a service by getting the boxes on the given page selected.

Terms https://www.cratejoy.com/sell/terms/
Privacy https://www.cratejoy.com/sell/privacy/

== Installation ==
1. Upload `cratejoy-subscription-box-affiliate-marketing.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget in Apperance > Widgets
